# 中文学习阅读材料

[English](README.md) | [中文](README_CN.md)

---

本项目旨在为非母语中文学习者提供主要通过阅读中国网络文学来学习中文的途径。

## 项目介绍

本项目基于"阅读是学习任何语言最有效的方式"的理念，为中文学习者提供简单、有趣、引人入胜的阅读材料——中国网络文学。这些材料按难度分级，从简单到困难，适合不同水平的学习者。

## 开始之前

在开始阅读之前，建议先测试一下您的中文词汇量：
https://www.arealme.com/chinese-vocabulary-size-test/cn/

## 词汇分级标准

根据教育部和国家语言文字工作委员会发布的《国际中文教育中文水平等级标准》(GF 0025-2021)，中文词汇分级体系如下：

| 等级 | 词汇量 | 累计词汇量 | 对应HSK等级 |
| ----- | --------- | --------------------- | ---------------------- |
| 初等水平一级 | 500词 | 500词 | HSK 1-2 |
| 初等水平二级 | 772词 | 1272词 | HSK 2-3 |
| 初等水平三级 | 973词 | 2245词 | HSK 3-4 |
| 中等水平四级 | 1000词 | 3245词 | HSK 4-5 |
| 中等水平五级 | 1071词 | 4316词 | HSK 5-6 |
| 中等水平六级 | 1140词 | 5456词 | HSK 6 |
| 高等水平七-九级 | 5636词 | 11092词 | HSK 7-9 |
