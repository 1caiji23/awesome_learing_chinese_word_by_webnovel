# 中文学习阅读材料

此项目为非中文母语者设计，主要通过阅读中国网络文学来学习中文。

## 项目简介

本项目基于"阅读是学习任何语言最有效的方法"这一理念，为中文学习者准备了轻松、有趣且引人入胜的阅读材料 和 中国网文。这些材料按照难度分级，从简单到困难，适合不同水平的学习者

## 开始使用

在开始阅读之前，建议先测试你的中文词汇量：
https://www.arealme.com/chinese-vocabulary-size-test/cn/

## 词汇量分级标准

根据教育部和国家语委发布的《国际中文教育中文水平等级标准》(GF 0025-2021)，中文词汇量分级体系如下：

| 等级       | 词汇量    | 累计词汇量   | 对应 HSK 级别 |
| -------- | ------ | ------- | --------- |
| 初等一级     | 500 词  | 500 词   | HSK 1-2 级 |
| 初等二级     | 772 词  | 1272 词  | HSK 2-3 级 |
| 初等三级     | 973 词  | 2245 词  | HSK 3-4 级 |
| 中等四级     | 1000 词 | 3245 词  | HSK 4-5 级 |
| 中等五级     | 1071 词 | 4316 词  | HSK 5-6 级 |
| 中等六级     | 1140 词 | 5456 词  | HSK 6 级   |
| 高等七 - 九级 | 5636 词 | 11092 词 | HSK 7-9 级 |
